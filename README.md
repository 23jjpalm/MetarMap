# METAR Map Updates

This repository hosts the stable update package for the Raspberry Pi METAR map.

Current stable version: `1.1.2`

The Raspberry Pis check:

`https://raw.githubusercontent.com/23jjpalm/MetarMap/main/updates/manifest.json`

For a future update, replace both files inside the `updates` folder in one commit:

- `updates/manifest.json`
- `updates/metar-map-update.tar.gz`
